# Foggerwala Project Coding Guidelines

This document outlines the rules and conventions for developers and AI assistants working on the Foggerwala project.

## Technology Stack
- **Framework**: React + Vite
- **Styling**: Vanilla CSS and CSS Modules (`*.module.css`)
- **Routing**: React Router

## Development Workflow
- Keep components modular and reusable.
- Use CSS Modules for page/component-specific styling to avoid class name collisions.
- Ensure layouts are responsive across mobile, tablet, and desktop screens.
- Keep comments and docstrings updated.

## AI Assistant Configuration
- Antigravity will automatically load these rules when this repository is opened in the IDE.
